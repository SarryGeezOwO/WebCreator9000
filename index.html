Automatic HTML, CSS, JS file generator

Web Creator 69 is a open-source tool that helps the user create a folder that automatically makes a HTML, CSS, JS files inside the given folder. This tool is open-source and mainly uses java on it's creation all it needs is a JAVA installed on your computer to be enable modification.Beware that this tool only runs on windows as support for Linux and MacOs is still on development.


Features

  • Opens the folder in file explorer upon creation.
  • HTML and CSS instantly gets created with a template upon folder creation.
  • Browsing a directory is now easy.
  • When the directory specified already exists, the tool will not proceed on creation.
  • Tool can be modify by the user in anyway without permissions needed.
  • Lightweight 133kb.
  • Installation proccess is quick and efficient.
  • It's Free! and may be used in commercial purposes.
